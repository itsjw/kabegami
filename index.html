Kabegami

Kabegami is a cross-distro wallpaper manager for Linux. Download the latest release here.

FAQ

What's up with the name? Kabegami (壁紙) means "wallpaper" in Japanese.

Why did you make it? I'm a wallpaper nut. I have a flobbity-jillion wallpapers stored on my computer. I obsess over picking out the perfect wallpaper to suit my mood, which seems important to me because I spend so much time on my computer every day. What others call obsessive compulsion I call feng shui. Anyway, the built-in wallpaper pickers for each desktop environment are passable, but they typically don't allow you to delete images immediately (instead, you have to use the file manager or CLI), there's no way to add metadata to images (such as tags), there's no way to group images into "playlists," and they don't keep track of which wallpapers you tend to prefer over long periods of time. I built Kabegami because I wanted to address all of these little problems. To be clear, Kabegami doesn't have all of those features yet, but that's where it's headed.

Couldn't you have just used a photo manager, like Shotwell or gThumb? Well, crap. I hadn't thought of that till just now. But even so, sometimes it's just fun to build something, you know?

Of what is it made? It was built using Electron, Vue, and Bulma.
